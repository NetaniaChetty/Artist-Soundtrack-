🎤 Taylor Swift Interactive Fan Website

A fully interactive and visually immersive fan website dedicated to Taylor Swift, built with modern front-end techniques and custom animations.

This project goes beyond a basic static site by incorporating dynamic effects, smooth animations, and engaging UI elements to create a rich user experience.

---

🚀 Key Features

🌌 Advanced Visual Effects
- Animated **starfield background** using Canvas API
- Dynamic wave animations with SVG
- Glassmorphism UI design
- Gradient lighting and layered backgrounds

🖱️ Interactivity & Animation
- Mouse-based **parallax movement**
- Scroll-triggered **reveal animations** (Intersection Observer)
- Animated CTA button with sparkle effects ✨
- Smooth hover transitions across UI components

🎵 Content & Layout
- Hero section with interactive elements
- Album timeline with cover images and eras
- Featured content panels (About + Events)
- Favorite songs section
- External links (Spotify, Store, Wikipedia)

📱 Responsive Design
- Fully responsive layout for mobile, tablet, and desktop
- Adaptive grid system

---

🛠️ Tech Stack

- **HTML5** – Structure and layout  
- **CSS3** – Custom properties, animations, glassmorphism UI  
- **JavaScript (Vanilla)** – Interactivity and animations  
- **Canvas API** – Starfield animation  
- **Intersection Observer API** – Scroll-based effects  
- **SVG** – Wave animations
